R package to compute age-standardized epidemiological rates in a tidy way.

R package to download and cache files from Zenodo deposits.

zenstats

R package to fetch statistics of views, downloads and volume from Zenodo deposits.

R package to compute counts and statistics of occurrence of events in sequences that meets a criterion.

R package to compute zonal statistics of climate indicators.

R package to retrieve municipal climate indicators of Brazilian municipalities.

zonalclim

R package with helper functions to compute zonal statistics using NetCDF and sf objects for climate studies.

R package to access the Plugfield API and retrieve data from weather stations.

R package to download dengue data from the OpenDengue project.
