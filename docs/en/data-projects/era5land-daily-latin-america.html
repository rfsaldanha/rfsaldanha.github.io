Introdução

A reanálise ERA5-Land, do Programa Copernicus, é uma excelente fonte de dados climáticos com cobertura global de áreas terrestres desde 1950 até o presente, em resolução espacial de 10 km. Os dados originais têm intervalo horário, e agregados mensais também estão disponíveis no Copernicus Data Store (CDS).

Para algumas aplicações, como modelagem de doenças sensíveis ao clima, o intervalo horário pode ser detalhado demais, enquanto a agregação mensal pode ser grosseira demais.

Por esse motivo, criei agregados diários de alguns indicadores ERA5-Land para algumas regiões.

Metodologia

Desenvolvi um script em R usando o pacote KrigR (Kusch and Davy 2022). O script baixa um conjunto de indicadores, a partir de 1950, para uma caixa delimitadora cobrindo a América Latina (coordenadas -118.47,-34.1,-56.65, 33.28) e agrega os dados de horários para diários, salvando os resultados como arquivos NetCDF. Cada arquivo resultante cobre um mês de um ano e apresenta camadas de dados para cada dia do respectivo mês.

A tabela abaixo contém as funções de agregação temporal aplicadas a cada indicador climático.
